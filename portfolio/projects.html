Wikipedia page replica

This project is a simple replica of a Wikipedia article page created using only basic HTML tags, without any CSS or JavaScript. It demonstrates how to structure a webpage using semantic elements such as headings, paragraphs, lists, hyperlinks, images, and sections. The layout is modeled after a typical Wikipedia article, focusing on clean, readable content organization. This project highlights an understanding of pure HTML and how web content can be meaningfully arranged without styling or interactivity.
